Connect your Amazon Ads account

AdScale connects directly to Amazon Ads via their official API. You'll need Amazon Ads API credentials — this is a one-time setup that gives you full control over your data.

1

Create an Amazon Ads API application

Go to advertising.amazon.com → Tools → API Access → Apply for credentials. You'll get a Client ID and Client Secret.

2

Enter your API credentials

Paste your Client ID and Client Secret below. These are stored locally — never shared.

3

Authorize with Amazon (OAuth)

Click below to open Amazon's login page and grant AdScale read/write access to your ad account.

Redirect URI: http://localhost:3000/auth/callback
4

Select your marketplace

Choose which Amazon marketplace to connect (US, UK, EU, etc.). You can add more marketplaces later.

Amazon Ads API · OAuth 2.0 · Read/Write Scope
